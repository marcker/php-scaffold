# PHP Scaffold

A PHP RESTful web application scaffold using [Flight](http://flightphp.com/).

## Install

```bash
git clone https://github.com/marcker/php-scaffold.git
cd php-scaffold/app
composer install
sudo php -S 0.0.0.0:3000
```
